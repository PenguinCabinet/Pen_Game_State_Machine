# The Pen Game Programing Finite State Machine

これは[GoのゲームライブラリEbiten用](https://ebiten.org/)のステートマシンライブラリです。
## 提供する機能
* ゲームプログラミングのためのスタック型のステートマシン
* 直前のフレームと今のフレーム間で経過した時間のdeltatime
